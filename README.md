# -OptiStock-Supply-Chain-Inventory-Optimization
📌 Project Overview

OptiStock is an end-to-end Supply Chain Analytics project designed to monitor inventory performance, identify inefficiencies, and support data-driven decision-making.

The dashboard provides real-time insights into:

Inventory health
Demand vs supply gaps
Supplier performance
Stock movement trends
🎯 Business Problem

Inefficient inventory management leads to:

Overstock (increased holding cost)
Stockouts (lost sales)
Poor demand planning

This project solves:

Where inventory is mismanaged
Which products need replenishment
How supplier delays impact stock
🛠️ Tech Stack
Power BI – Dashboard & Visualization
Power Query – Data Cleaning & Transformation
DAX – KPI Calculations
Excel (CSV) – Data Source
📊 Key Analysis Performed
1. Inventory Health Analysis
Identified:
Overstock items
Stockout risks
Categorized inventory based on availability
2. Demand vs Supply Analysis
Compared demand with actual supply
Highlighted mismatch trends
Identified understock/overstock scenarios
3. Reorder Point Monitoring
Calculated reorder levels
Flagged products requiring immediate restock
4. Supplier Performance Analysis
Measured:
Lead time
Delivery consistency
Identified slow or unreliable suppliers
5. Inventory Aging Analysis
Tracked how long stock remains unsold
Highlighted dead stock and slow-moving items
📈 Key KPIs Tracked
Inventory Turnover Ratio
Stockout Rate (%)
Order Fulfillment Rate
Average Lead Time
Overstock Value
📊 Dashboard Features
Interactive filters (product, warehouse, time)
Drill-down analysis for detailed insights
KPI cards with real-time metrics
Visual comparison of demand vs supply
💡 Key Insights
⚠️ Certain products show consistent overstocking
🚨 Stockout risk observed in high-demand items
⏳ Supplier delays directly impact inventory gaps
📦 Slow-moving inventory increases holding cost
💼 Business Recommendations
Optimize reorder strategy using demand trends
Reduce overstock by improving forecasting
Replace or negotiate with low-performing suppliers
Focus on fast-moving products for better ROI
